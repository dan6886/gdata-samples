YouTube Topic Explorer

YouTube Topic Explorer is a sample application written to demonstrate support for Topics in the YouTube Data API v3. It makes use of the Freebase API to search for topics related to a specific term, and to translate in between topic ids and descriptive phrases.

You can use the Topic Explorer without logging in, in which case you will be able to search for YouTube videos, channels, and playlists related to a given topic, but you won't be able to favorite, like, or subscribe to them. Additionally, if you log in, you will see a list of recommended topics based on your account's likes, favorites, and watch later playlists.

This is an open source project, and feel free to look at the source code for examples of calling the YouTube and Freebase APIs. It's built on the AngularJS framework, using the Yeoman toolkit for managing dependencies and the build environment.
